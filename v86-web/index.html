AurionOS in your browser

This runs a full operating system through in-browser emulation. It works, but it is noticeably slower than the real thing - that is just what happens when you emulate hardware in JavaScript.

For the real experience, run AurionOS in VirtualBox or on actual hardware. It is significantly faster and you will see what this system can actually do.

What makes AurionOS different
Built from scratch
Not a Linux distro. Not a fork. Every line of kernel code was written from the ground up, with full control over the architecture.
Lightweight and fast
Minimal footprint. Boots in seconds on real hardware. Every component earns its place - no bloat, no unnecessary layers.
Open source
Read every line. Build it yourself. Contribute improvements. The full source is on GitHub - nothing hidden, nothing proprietary.
Run it for real

VirtualBox

The quickest way to try AurionOS at full speed. Download the ISO, create a new VM with 512MB RAM, attach the ISO as a CD drive, and boot.

Type: Other/Unknown RAM: 512 MB Storage: Optional

Build from source

Clone the repository and build AurionOS yourself. You will need a cross-compiler toolchain and NASM. Full instructions are in the README.

git clone https://github.com/Luka12-dev/AurionOS.git cd AurionOS make all
